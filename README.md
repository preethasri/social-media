**Introduction -**
A Social Media App for people to connect and chat


**Features -**

- Create a post
  - Like  
  - Comments    
- Edit & delete post
- User feed   
  - Sort by latest, oldest & trending 
- User profile
  - Add profile picture
  - Bio
  - Portfolio URL
- Follow/Unfollow
- Explore feed
- Bookmark post
- Authentication
  - Sign-up
  - Login
  - Logout

## **Tech Stack -**

- React JS
- Redux toolkit + React-Redux
- Tailwind CSS
